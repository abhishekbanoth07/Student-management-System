Student Management System

This is a full-stack data-driven web application developed using Python, MySQL, and Streamlit.

The application allows users to securely manage student records through a simple and interactive interface. It includes authentication, complete CRUD operations, search functionality, and basic data analytics.

🔹 Features:
- User Signup & Login (Authentication system)
- Add, View, Update, Delete student records
- Search students by name
- Data analytics (marks distribution, course-wise averages, topper)
- CSV import/export support (optional if added)

🔹 Tech Stack:
- Frontend: Streamlit
- Backend: Python
- Database: MySQL
- Libraries: Pandas, mysql-connector-python

🔹 Architecture:
- app.py → Handles UI and user interaction
- operations.py → Handles all database operations (SQL queries)
- db.py → Manages database connection

This project demonstrates separation of concerns, clean architecture, and practical full-stack development skills.
